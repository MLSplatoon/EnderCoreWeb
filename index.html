Sobre Ender Core
Hecho para la comunidad

Hecho por un apasionado por la tecnología y la creatividad. Que disfruta haciendo proyectos en programación, para jugar y disfrutar sin ninguna interrupción

Server
Varios Modos en un solo servidor.

Ender Core cuenta con varios modos en el servidor, Skywars, Bedwars, Survival y Muchos más. Hechos 100% a mano para tener variedad de modos y no aburrirte. Todo fácil con /menu usando un solo click.

Desarrollado para la comunidad

Este servidor está hecho para la comunidad y por la comunidad, aceptamos sugerencias, ideas, y ayudas. Ender Core no sería lo que es hoy en dia sin los jugadores que juegan en el servidor.

Plugins y funcionamiento

Ender Core cuenta con varios plugins que se usan para garantizar una mejor experiencia y funcionalidad en el servidor, Usamos PaperMc con la versión de Minecraft 1.21. También tenemos el Servidor en Github lo cual nos permite compartirlo y hacer que sea de Código abierto.
